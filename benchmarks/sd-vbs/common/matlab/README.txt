This folder contains Matlab implementation of the common functions used across the suite.
The currently supported functions include:

read_image8u_bmp.m : 	This function is similar to imread() in Matlab. It supports file read of a .bmp file.
write_image8u_bmp.m : 	This function is similar to imwrite() in Matlab. It supports file read of a .bmp file.
reshapeMatrix.m : 	Similar to reshape(X, SIZ) built-in Matlab function
product.m :		Similar to prod(X, DIM) built-in Matlab function
